# phalanx
<img src="./fog.png" width="200">
phalanx is a password generator with any file.

### Usage
**install**
```bash
pip3 install git+https://github.com/TakutoYoshikai/phalanx.git
```

**generate password**
```bash
phalanx <ANY FILE> -p <MASTER PASSWORD> -l <PASSWORD LENGTH>
```
